# financial-digital-twin
Financial Digital Twin (FDT) is a fintech system that creates a virtual model of a user’s financial life and simulates future financial outcomes based on real-time data and user-defined scenarios.

The project introduces a unique concept called "Antigravity Economics", where traditional financial constraints such as debt pressure and risk limitations are inverted or removed to analyze alternative financial behaviors.

This system enables users to:
- Track their financial state over time
- Simulate different financial decisions
- Compare normal vs constraint-free (antigravity) scenarios
- Understand future financial risks and outcomes

The project is built using a step-by-step GitHub workflow with strong DBMS integration, including schema design, triggers, and simulation logic.
